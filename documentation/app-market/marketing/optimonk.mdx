## Optimonk

*Note: Click [here](mobile-admin-link) to navigate to the mobile admin app version of this article*

Optimonk allows you to create a personalized website experience for your website visitors easily, thus maximizing conversions.

Here is how you install Optimonk on your Aasaan website.

**STEP 1:**
Login to your Optimonk dashboard and click on the Install Optimonk button.

**STEP 2:**
Copy the account ID from the code as shown.

**STEP 3:**
Install Optimonk from the app market. Click [here](install-link) to learn how to install apps from your app market.

**STEP 4:**
Paste the account ID from step 2.

**STEP 5:**
Click on Verify.
You have added Optimonk to your website.
