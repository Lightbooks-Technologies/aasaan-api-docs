Wheelofpopups helps you create effective spin-to-win website popups to boost visitor engagement and drastically reduce cart abandonment.

*Note: Make sure to install Wheelofpopups from the aasaan app market. Click [here](install-app-link) to learn how to install apps from your app market.*

**STEP 1:**  
Login to your Wheelofpopups dashboard and click on the promotion you want to set up.

**STEP 2:**  
Click on Publish.

**STEP 3:**  
Copy the part of the code between `api/` and `/widget.js`, as shown below.

**STEP 4:**  
Go to your installed Wheelofpopups app in aasaan. Paste the code you copied from step 3 and click on Verify.  
The status should say "Active".

**STEP 5:**  
Go back to your Wheelofpopups dashboard and click on Retry to verify that the code has been installed.  
You have successfully set up your Wheelofpopups promotion.
