## Borzo

*Integrate Borzo Delivery with Your Store*  
*Set up Borzo on your Store for Local Shipping*

*Note: Click [here](mobile-admin-link) to navigate to the mobile admin app version of this article*

You can watch the video version of this article [here](video-link).

Borzo is a fast intracity courier delivery service that makes it easier for businesses to ship locally. Aasaan offers a very easy integration with Borzo to manage your local shipping needs. Here's how you integrate Borzo to your store.

*Note: Make sure to install Borzo from the app market. Click [here](install-app-link) to learn how to install apps from your app market.*

### Part 1: Obtaining the Borzo API Key

### STEP 1:

1. Login to your Borzo account and click on Integration.
2. If using a newly created Borzo account, the first two steps shown in the image below - executing two orders in the Borzo test environment - must be completed before the live API key can be requested. If the Borzo account has been used before, skip step 2 and proceed directly to step 3. Click [here](borzo-account-link) to learn how to complete the steps shown in a new Borzo account to get the live API key.
   
### STEP 3:

3. To obtain the live API key, enter the email address you wish the key to be sent to.
4. Click on Get API Token.
   The API Token has been requested, and will appear on your dashboard within 24 hours.
   Once the live API key has appeared on the integration page of your Borzo dashboard, click on Copy Token.

### Part 2: Verifying the Borzo API key

### STEP 5:

5. Go to your installed Borzo app in aasaan. Paste the API token obtained in step 4, in the field shown below and click on Verify.

You have successfully integrated Borzo delivery to your store.
