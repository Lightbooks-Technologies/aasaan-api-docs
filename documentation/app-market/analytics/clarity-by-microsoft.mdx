## Clarity by Microsoft

**Set up Clarity by Microsoft on Your Store**  
*Add Microsoft Clarity to your Store*  
*Note: Click here to navigate to the mobile admin app version of this article*

You can watch the video version of this article [here](video-link).

Microsoft Clarity is a free analytics tool built to help website managers improve their website experiences by better understanding site visitors' behaviour. When you add Clarity on your website, you get access to various forms of data such as visitor session recordings and heatmaps to understand your website visitors' behaviour.

If you have an existing account in Clarity by Microsoft and you wish to track your website analytics, here's how you set it up on aasaan.  
*Note: Make sure to install Clarity by Microsoft from the app market. Click [here](app-market-link) to learn how to install apps from the app market.*

### STEP 1:

Login to your Clarity by Microsoft account.

### STEP 2:

Click on the settings button of the project you wish to use for your website.

### STEP 3:

Copy the given project ID.

### STEP 4:

Go to your Microsoft Clarity app in aasaan, paste the project ID and click on Verify.  
You have successfully set up Clarity on your website.
