## Bing Webmaster

**Verify your Website on Bing Webmaster**  
*Note: Click here to navigate to the mobile admin app version of this article*

Bing Webmaster is a free tool offered by Microsoft which allows you to verify and index your website on the Bing search engine. aasaan provides an easy solution to verify your website with Bing webmaster. Here's how you do it:

*Note: Make sure to Install Bing Webmaster from the app market.*  
*Click [here](app-market-link) to learn how to install apps from the aasaan app market.*

### STEP 1:

Login to Bing Webmaster.

### STEP 2:

Enter your website URL.

### STEP 3:

Click on Add.

### STEP 4:

Choose your verification method as HTML Meta Tag.

### STEP 5:

Copy the code snippet that comes after `content=`. It is marked below for your reference.

### STEP 6:

Go to your Bing Webmaster app in aasaan, and paste the code snippet from Step 5. Then click on Verify.  
Your Bing Webmaster code has been successfully added to your website.

### STEP 8:

Go back to your Bing Webmaster dashboard and click on Verify.  
Your website has been verified on Bing Webmaster.
