## Body Color

**How to Add Body Color**

Aasaan offers an exciting array of options to personalize your website's body color, including background, text, heading, link, button, and icon/arrow colors. This empowers you to create a website that perfectly aligns and enhances the user experience by emphasizing readability, visual appeal, and cohesive design.

### STEP 1:
Log in to your aasaan desktop admin panel and click on Storefront.

### STEP 2:
Click on the Content Template in which you wish to apply the Body Color. You can also click on Create Template if you wish to create a new template for your website.

### STEP 3:
Click on Theme.

### STEP 4:
Click on the Body Color drop down.

### STEP 5:
Click on Background Color to add the background color for the website.

### STEP 6:
Choose the body background color from the color palette or enter the color code manually. The changes will be reflected on the background of your storefront. Please refer to fig: Background color.

![Background color](https://cdn.aasaan.app/docs/body/body_1.png)

### STEP 7:
Click on Text Color to add foreground color for text on background colors.

*Note: Text Color will reflect on foreground color for text on the background, for example, Search, MSRP, Reviews Counts, SKU number, etc., on your website.*

### STEP 8:
Choose the text color for the body from the color palette or enter the color code manually. The changes will be reflected on the text on the background of your storefront. Please refer to fig: Text color.

![Text color](https://cdn.aasaan.app/docs/body/body_2.png)

### STEP 9:
Click on Heading Color option to preset color for titles and headings on your website.

*Note: Heading Color will reflect on all the titles and headings, for example, widgets, footer, and side menu headings on your website.*

### STEP 10:
Choose the heading color for the Body from the color palette or enter the color code manually. The changes will be reflected on the titles and headings on your website. Please refer to fig: Heading color.

![Heading color](https://cdn.aasaan.app/docs/body/body_3.png)

### STEP 11:
Click on Link Color to select the current color for clickable text (hyperlinks) on a website.

### STEP 12:
Choose the link color from the color palette or enter the color code manually. The changes will be reflected on clickable text (hyperlinks) on a website. Please refer to fig: Link color.

![Link color](https://cdn.aasaan.app/docs/body/body_4.png)

### STEP 13:
Click on Link Hover/Active Color for changing the color of clickable text when hovering or clicking.

*Note: Hover color refers to the change in color of an element (such as a button or link) when a user hovers their cursor over it.*

### STEP 14:
Choose the link hover/active color from the color palette or enter the color code manually. The changes will be reflected on clickable text (hyperlinks) on a website when hovering or clicking. Please refer to fig: Link hover color.

![Link hover color](https://cdn.aasaan.app/docs/body/body_5.png)

### STEP 15:
Click on Breadcrumb Color for displaying the color of the navigation trail on a website.

*Note: A breadcrumb is a user interface element that displays a trail of clickable links. It allows users to easily retrace their steps and navigate back to previous pages or sections.*

### STEP 16:
Choose the breadcrumb color from the color palette or enter the color code manually. The changes will be reflected on Breadcrumb on a website. Please refer to fig: Breadcrumb color.

![Breadcrumb color](https://cdn.aasaan.app/docs/body/body_6.png)

### STEP 17:
Click on the Breadcrumb Hover Color for showing the color when hovering over a navigation trail link.

### STEP 18:
Choose the breadcrumb hover color from the color palette or enter the color code manually.

### STEP 19:
Click on Breadcrumb Active Color for indicating the color of the current or active navigation trail link.

### STEP 20:
Choose the breadcrumb active color from the color palette or enter the color code manually. The changes will be reflected on the current or active navigation trail link website. Please refer to fig: Breadcrumb active color.

![Breadcrumb active color](https://cdn.aasaan.app/docs/body/body_7.png)

### STEP 21:
Click on Success Color. It is used to indicate positive or successful outcomes or elements.

### STEP 22:
Choose the success color from the color palette or enter the color code manually. The changes will be reflected on successful outcomes on the website (for example, product has been added to your cart). Please refer to fig: Success color.

![Success color](https://cdn.aasaan.app/docs/body/body_8.png)

### STEP 23:
Click on Success Background Color. It is used to highlight the background of elements with successful outcomes.

### STEP 24:
Choose the success background color from the color palette or enter the color code manually. The changes will be reflected on the background of successful outcomes on the website (for example, product has been added to your cart). Please refer to fig: Success background color.

![Success background color](https://cdn.aasaan.app/docs/body/body_9.png)

### STEP 25:
**Button Color:**

Click on the Text Color to set the color of the text within a button.

### STEP 26:
Choose the text color from the color palette or enter the color code manually. The changes will be reflected on the text of the button on your website. Please refer to fig: Button text color.

![Button text color](https://cdn.aasaan.app/docs/body/body_10.png)

### STEP 27:
Click on Border Color to define the color of the button's border.

### STEP 28:
Choose the border color from the color palette or enter the color code manually. The changes will be reflected on the border of the button on your website. Please refer to fig: Border color.

![Border color](https://cdn.aasaan.app/docs/body/body_11.png)

### STEP 29:
Click on Background Color to specify the background color of a button.

### STEP 30:
Choose the background color from the color palette or enter the color code manually. The changes will be reflected on the background of the button on your website. Please refer to fig: Button background color.

![Button background color](https://cdn.aasaan.app/docs/body/body_12.png)

### STEP 31:
Click on Hover / Active Color to indicate the color change of a button when hovered over or clicked.

### STEP 32:
Choose the hover / active color from the color palette or enter the color code manually. The changes will be reflected on the text on the button when hovering or clicking. Your website. Please refer to fig: Hover/Active color.

![Hover/Active color](https://cdn.aasaan.app/docs/body/body_13.png)

### STEP 33:
Click on Hover / Active Border Color to define the border color of a button when hovered over or clicked.

### STEP 34:
Choose the hover / active border color from the color palette or enter the color code manually. The changes will be reflected on the border of the button when hovering or clicking. Your website. Please refer to fig: Hover/Active border color.

![Hover/Active border color](https://cdn.aasaan.app/docs/body/body_14.png)

### STEP 35:
Click on Hover / Active Background Color to set the background color of a button when hovered over or clicked.

### STEP 36:
Choose the hover / active background color from the color palette or enter the color code manually. The changes will be reflected on the background of the button when hovering or clicking on your website. Please refer to fig: Hover/Active background color.

![Hover/Active background color](https://cdn.aasaan.app/docs/body/body_15.png)

### STEP 37:
**Icon/Arrow Color**

Click on Color. This is used to specify the color of icons or arrow graphics.

*Note: Icons on websites are compact, visual symbols or representations used to convey information or actions. Arrows on websites are graphical elements in the shape of pointed lines or triangles, guide user actions, or highlight clickable elements.*

### STEP 38:
Choose the color from the color palette or enter the color code manually. The changes will be reflected on Icons your website. Please refer to fig: Icon color.

![Icon color](https://cdn.aasaan.app/docs/body/body_16.png)

### STEP 39:
Click on Background Color. It is used to set the background color behind icons or arrows.

### STEP 40:
Choose the background color from the color palette or enter the color code manually. The changes will be reflected on the background of Icon on the website. Please refer to fig: Icon background color.

![Icon background color](https://cdn.aasaan.app/docs/body/body_17.png)

### STEP 41:
Click on Hover/Active color. It is used for indicating the color change when icons or arrows are hovered over or clicked.

### STEP 42:
Choose the hover/active color from the color palette or enter the color code manually. The changes will be reflected on Icon on the website when hovering or clicking. Please refer to fig: Icon hover color.

![Icon hover color](https://cdn.aasaan.app/docs/body/body_18.png)

### STEP 43:
Click on Hover/Active Background color. It is used to define the background color when icons or arrows are hovered over or clicked.

### STEP 44:
Choose the hover/active background color from the color palette or enter the color code manually. The changes will be reflected on the background of Icon on the website when hovering or clicking. Please refer to fig: Icon hover/active background color.

![Icon hover/active background color](https://cdn.aasaan.app/docs/body/body_19.png)

### STEP 45:
Click on Save.

Your body color settings have been saved successfully.

### STEP 46:
Click on Publish to make your changes live on your website.

Your body color changes have been successfully published.
